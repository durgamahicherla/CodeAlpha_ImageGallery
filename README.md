# Modern Image Gallery

A beautiful and responsive image gallery created using HTML, CSS, and JavaScript.

## Features
- Responsive gallery layout
- Category filter buttons
- Lightbox image preview
- Smooth hover zoom effects
- Modern dark UI design

## Categories Included
- Nature
- City
- Animals
- Cars
- Space

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Preview
This project provides a stylish image gallery experience with filtering and fullscreen image viewing.
